## 🏁 How to Use This Repo

1. **Start each day** by reading the next section in `ROADMAP.md`
2. **Study** the linked resources and take notes in `notes/`
3. **Code** — push every project/exercise to `projects/` or `challenges/`
4. **Log** your progress in `daily-log/month-XX/day-XXX.md`
5. **Post** on X using templates from `x-posts/templates.md`
6. **Commit & push** every day — your GitHub graph is your proof of work
