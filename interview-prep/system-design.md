# 🏗️ Web3 System Design — Interview Prep

## Design Problems

### 1. Design a Decentralized Exchange (DEX)
- AMM vs order book trade-offs
- Liquidity pool design
- Price oracle integration
- Front-running protection
- Multi-chain deployment

### 2. Design a Lending Protocol
- Collateralization ratios
- Interest rate models
- Liquidation mechanisms
- Oracle dependencies
- Risk parameters

### 3. Design an NFT Marketplace
- On-chain vs off-chain metadata
- Royalty enforcement (ERC-2981)
- Auction mechanisms
- Indexing strategy (The Graph)
- IPFS vs Arweave for storage

### 4. Design a DAO Governance System
- Token-weighted vs quadratic voting
- Proposal lifecycle
- Timelock security
- Delegate systems
- Off-chain voting (Snapshot) vs on-chain

### 5. Design a Cross-Chain Bridge
- Trust assumptions
- Message verification
- Finality considerations
- Security considerations (bridge hacks)
- Liquidity management
