# 🎯 Solidity Interview Questions

## Beginner

1. **What is the difference between `memory`, `storage`, and `calldata`?**
   - *storage*: persistent on-chain, expensive
   - *memory*: temporary during function execution
   - *calldata*: read-only function input, cheapest
